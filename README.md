## About

**Equity2Bitcoin Consulting LLC** is a milestone-based educational consulting firm founded by **Brendan Ngehsi Ngwa Nforbi** on November 3, 2025 in California, USA.

**Mission:** To help homeowners responsibly unlock dormant home equity and reposition it into Bitcoin through a compliant, milestone-based consulting model that preserves regulatory freedom, client control, and long-term wealth autonomy.

**Model:** A 30% milestone success fee on equity extracted — triggered only at verified loan approval (Phase 3). All other phases are free. The firm never handles funds, arranges loans, or makes investment recommendations.

**Example:** $300,000 equity extraction → **$90,000 consulting fee**

## Deployment

Deployment is fully automated via GitHub Actions:

1. Push to `main` triggers the `Deploy to GitHub Pages` workflow
2. Actions installs dependencies (`npm ci`) and runs `npm run build`
3. The `dist/` folder is uploaded and deployed to GitHub Pages
4. Live at: **https://newanforbi.github.io/Equity2Bitcoin-Business-Plan/**
